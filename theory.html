Patterns Créationnels

Factory Method

But : Déléguer la création d'objets aux sous-classes.

Indices : Classe abstraite avec méthode abstraite de création, return new ConcreteProduct

Singleton

But : Une seule instance pour toute l'application.

Indices : Constructeur private, getInstance(), instance static

Patterns Comportementaux

Chain of Responsibility

But : Passer une requête le long d'une chaîne de handlers.

Indices : Attribut successeur, if traite else passe au suivant, null à la fin

Command

But : Encapsuler une action dans un objet.

Indices : Interface Command, execute(), liste de commandes, exécution différée

Observer

But : Notifier automatiquement des objets quand un état change.

Indices : attach(), detach(), notify(), Liste d'observers

State

But : Changer le comportement selon l'état.

Indices : Attribut state, délégation state.method(), changement d'état

Strategy

But : Algorithmes interchangeables.

Indices : Attribut strategy, injection dans constructeur
