Description : Implémentation des patterns Abstract Factory et Factory Method pour illustrer la création d'objets et de familles d'objets liés.

Patterns utilisés : Abstract Factory, Factory Method

Abstract Factory Pattern

Théorie

Comment le reconnaître ?

Je reconnais Abstract Factory grâce à une interface ou classe abstraite qui définit plusieurs méthodes de création (ex : createProductA(), createProductB()). Chaque ConcreteFactory implémente ces méthodes pour créer une famille d'objets liés.

Exercice

Factory Method Pattern

Théorie

Comment le reconnaître ?

Je reconnais Factory Method car c'est un cas particulier de Template Method où la méthode abstraite fait un new. Il y a une classe abstraite avec une méthode abstraite de création, et les ConcreteCreator implémentent en faisant return new ConcreteProduct().
